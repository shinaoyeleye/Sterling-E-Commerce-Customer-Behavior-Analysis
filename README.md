# Sterling E-Commerce Customer Behavior Analysis Dashboard (Power BI)
# Project Overview
This project analyzes e-commerce transaction data to understand customer shopping behavior, demand patterns, and business performance using Microsoft Power BI.
The objective was to transform raw transaction data into an interactive dashboard that supports data-driven decision-making and provides insights into purchasing trends, payment preferences, and location-based performance.
# Project Objectives
•	Analyze customer purchasing behavior
•	Identify high-demand product categories
•	Examine customer payment method preferences
•	Evaluate order distribution across cities and locations
•	Measure customer engagement and repeat purchases
•	Analyze demand trends over time
# Tools Used
•	Microsoft Power BI
•	Power Query
•	DAX (Data Analysis Expressions)
•	Microsoft Excel
# Dashboard Features
•	KPI Cards (Total Orders, Total Customers, Total Quantity Ordered, Orders per Customer)
•	Product category demand analysis
•	Payment method distribution chart
•	City / location performance comparison
•	Demand trend line chart over time
•	Interactive date slicer for time-based filtering
•	Supporting data table for detailed exploration
# Key Insights
•	Customer demand is concentrated in specific product categories.
•	Digital payment methods are widely preferred by customers.
•	Order volume is higher in major urban locations.
•	Customers demonstrate repeat purchasing behavior.
•	Demand fluctuates over time, indicating changing purchasing patterns.
